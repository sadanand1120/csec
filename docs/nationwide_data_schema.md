This is the first-pass schema for nationwide generated data. It is intentionally
source-oriented so each stage can be reviewed before app integration.

## Later Per-City Model Record

Each city will eventually add these sections:

- `geography`: county mapping, CBSA mapping, computation county, mapping
  confidence, and ambiguity notes.
- `housing`: HUD FMR area, one-bedroom monthly FMR, annual rent, vintage, and
  mapping confidence.
- `prices`: BEA RPP values for all items, goods, housing, utilities, and other
  services.
- `tax_curve`: gross grid, tax components, net income, PolicyEngine version,
  tax year, state/county/local flags.
- `cost_basket`: housing, food, transportation, healthcare, internet/mobile,
  other non-housing, and total.
- `provenance`: source URLs, file checksums, source rows/keys, transform script
  version, retrieval timestamp, and QA status.
